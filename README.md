# Coding-Prashant-JHA

Beginner-friendly Python practice programs and notes, organized by day so each topic stays small and easy to revisit.

## Project Layout

```text
Coding-Prashant-JHA/
├── README.md
├── DAY1/
└── DAY2/
```

## What’s Inside

### DAY1

Core Python basics and beginner exercises:

- variables and data types
- `id()` and object identity
- type conversion
- membership and identity operators
- input handling and conditions
- ASCII-based character checks
- nested `if` logic
- simple arithmetic practice

### DAY2

Collection and loop practice:

- lists and list traversal
- slicing
- reversing with loops
- duplicate removal
- palindrome checks
- string formatting
- tuples and immutability
- aliasing notes

## How to Run

Run any file from the repository root:

```bash

```

Some programs expect terminal input, while others print output directly.

## Purpose

This repo is a personal learning log for Python fundamentals. The files are intentionally small so each concept is easy to understand, run, and extend.
